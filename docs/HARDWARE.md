# 🛠 하드웨어 구성

## Raspberry Pi 기반 시스템

- **Raspberry Pi 4** - 메인 컨트롤러
- **MCP23017 x2** (I2C 0x20, 0x21) - 32개 GPIO 확장
- **ADS1115** (I2C 0x48) - 4채널 16-bit ADC (센서 읽기, 전압 모드 0~5V)
- **RTC DS1307** (I2C 0x68) - 실시간 시계
- **RS-485 토양 수분 센서 x12** (Modbus RTU) - 수분·온도·EC 측정
- **대기 온습도 센서** (구상 중) - 비닐하우스 내부 환경 모니터링 (개수 미정)

## 전원 및 릴레이 시스템

- **RD-125B** (DC 5V 10A + DC 24V 5A 듀얼 출력 SMPS)
  - CH1 (5V): 6채널 릴레이 보드 4개 VCC 공급
  - CH2 (24V): RS-485 토양 센서 12개 + 수위 센서 + MAX485 모듈 전원
- **AC 24V 트랜스포머 25W** - AC 24V 전자밸브 구동 (릴레이 접점 경유)
- **SSR 40A + 방열판** (구매 예정) - 관수펌프 (220V AC) 제어용
- 6채널 릴레이 모듈 x4 (총 24채널)
- 릴레이 구성: 구역 밸브 12개 + 펌프 1개 + 호스건 1개 + 급수/양액/체크 등 제어

> ⚠️ 라즈베리파이 GPIO 5V 핀으로 릴레이 보드 직접 구동 금지 — RD-125B CH1(5V) 외부 전원 사용

## 제어 신호 흐름

```
MCP23017 (3.3V GPIO 신호)
  └─ 6채널 릴레이 보드 IN 핀
       ├─ [전자밸브 채널] 릴레이 COM/NO → AC 24V 전자밸브
       └─ [펌프 채널] 릴레이 NO → SSR DC 입력 (+24V)
                                   └─ SSR AC 출력 → 220V AC 관수펌프
```

- **전자밸브 제어**: MCP 신호 → 릴레이 → AC 24V 전자밸브 직결
- **펌프 제어 (고전력)**: MCP 신호 → 릴레이 → SSR 40A DC 입력 → SSR AC 출력 → 220V 펌프
  - 릴레이 NO 단자 ← SMPS 24V+, COM → SSR DC IN (+)
  - 릴레이 ON 시에만 SSR에 24V 인가 → 펌프 회로 도통
  - SSR 입력: DC 3~32V 허용 (24V 사용), 출력: 220V AC / 40A
  - **방열판 필수**: 정격 10A 펌프 → SSR 발열 약 15W → 알루미늄 방열판 100×100mm 이상 + 서멀 그리스

## 전원 구성

| 전원 | 규격 | 공급 대상 | 비고 |
|------|------|---------|------|
| USB 어댑터 | DC 5V 3.1A | Raspberry Pi 4 | 전용 어댑터 (별도) |
| **RD-125B CH1** | **DC 5V 최대 10A** | 6채널 릴레이 보드 × 4 VCC | RPi GPIO 5V 핀 사용 금지 |
| **RD-125B CH2** | **DC 24V 최대 5A** | 토양 센서 12개, 수위 센서, MAX485 | 총 소비 약 1.5A |
| **AC 24V 트랜스 25W** | **AC 24V 25VA** | AC 24V 전자밸브 | 릴레이 접점 경유, DC와 절연 |

**GND 공통 연결 (필수):**
- Raspberry Pi GND ↔ RD-125B CH1 (–) ↔ RD-125B CH2 (–) ↔ MAX485 GND ↔ 수위 센서 신호 GND
- AC 24V 트랜스 접지는 DC 공통 GND에 연결 금지 (절연 유지)

## 배관 구성

- 지하수 -> 양액밸브 -> 양액탱크 (양액 보충 라인)
- 지하수 -> 급수밸브 -> 물탱크 -> 체크밸브 -> 관수펌프 -> 구역 1~12 (자동관수 메인 라인)
- 지하수 -> 호스건밸브 -> 호스건 (수동 호스건 독립 라인, 펌프 미사용)

호스건은 지하수 직결 독립 배관으로 관수펌프 회로와 완전 분리됨.
호스건 동작 중 자동관수 동시 실행 가능 (하드웨어 충돌 없음).

## 수위 센서 배선 (QDY30A, 0~5V 출력)

수위 센서 QDY30A (0~5V 아날로그 출력) 2개를 ADS1115 ADC를 통해 읽음.

| 센서 선색 | 연결 대상 | 비고 |
|---------|---------|------|
| 갈색 (전원+) | RD-125B CH2 DC 24V (+) | 센서 구동 전압 |
| 검정 (GND) | 공통 GND 버스 | DC 공통 GND |
| 파랑 (신호, 0~5V) | 전압 분배 회로 → ADS1115 AINx | 5V → 2.5V 변환 필요 |

**⚠️ 전압 분배 회로 필수** (ADS1115 최대 입력 3.6V 보호):

```
센서 신호선(파랑) ─[10kΩ]─┬─ ADS1115 AINx
                          [10kΩ]
                           │
                          GND
```

- 탱크 1 (물탱크): ADS1115 AIN0
- 탱크 2 (양액탱크): ADS1115 AIN1
- 캘리브레이션: `config/sensor_calibration.json` — 실제 탱크 수위 기준 empty/full 전압 실측 후 업데이트

## RS-485 토양 센서 배선 (CWT-Soil-THC-S, 12개)

MAX485 모듈 1개로 12개 센서를 병렬 연결 (멀티드롭 RS-485 버스).

**MAX485 ↔ Raspberry Pi GPIO 연결:**

| MAX485 핀 | Raspberry Pi GPIO | 역할 |
|----------|-----------------|------|
| DI | GPIO14 (TXD) | 데이터 송신 |
| RO | GPIO15 (RXD) | 데이터 수신 |
| DE + RE (묶음) | GPIO18 | 송/수신 방향 제어 |
| VCC | 3.3V 또는 5V | 모듈 전원 |
| GND | 공통 GND 버스 | - |

**센서 배선 (4가닥, 12개 병렬):**

| 선색 | 연결 대상 | 비고 |
|------|---------|------|
| 갈색 (VCC) | RD-125B CH2 DC 24V (+) | 5~30V 허용 |
| 검정 (GND) | 공통 GND 버스 | - |
| 노랑 (RS-485 A+) | MAX485 A 단자 | 버스에 병렬 연결 |
| 파랑 (RS-485 B-) | MAX485 B 단자 | 버스에 병렬 연결 |

**통신 설정:** 4800bps, No parity, 8 data bits, 1 stop bit
**Modbus 주소:** 각 센서 고유 주소 1~12 (`config/soil_sensors.json`에서 관리)

## 관수 펌프

- **모델**: 한일펌프 PA-1688SS
- **전원**: 단상 220V AC
- **정격 출력**: 2HP (1.5kW), 정격 전류 약 8~10A
- **제어**: SSR 40A (방열판 포함) → 릴레이 → MCP23017 pump 채널 (0x20 PA0)
- **주의**: 기동 돌입전류 약 50~70A → SSR 40A + 방열판 필수
