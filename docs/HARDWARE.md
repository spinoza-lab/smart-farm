# 🛠 하드웨어 구성

## Raspberry Pi 기반 시스템

- **Raspberry Pi 4** - 메인 컨트롤러
- **MCP23017 x2** (I2C 0x20, 0x21) - 32개 GPIO 확장
- **ADS1115** (I2C 0x48) - 4채널 16-bit ADC (센서 읽기, 전압 모드 0~5V)
- **RTC DS1307** (I2C 0x68) - 실시간 시계
- **RS-485 토양 수분 센서 x12** (Modbus RTU) - 수분·온도·EC 측정

## 전원 및 릴레이 시스템

- 릴레이 모듈 6개 x 4채널 (총 24채널)
- 50A 릴레이 x3개
- DC 5V 5A 전원 공급
- 릴레이 구성: 구역 밸브 12개 + 펌프 1개 + 호스건 1개

## 배관 구성

- 지하수 -> 양액밸브 -> 양액탱크 (양액 보충 라인)
- 지하수 -> 급수밸브 -> 물탱크 -> 체크밸브 -> 관수펌프 -> 구역 1~12 (자동관수 메인 라인)
- 지하수 -> 호스건밸브 -> 호스건 (수동 호스건 독립 라인, 펌프 미사용)

호스건은 지하수 직결 독립 배관으로 관수펌프 회로와 완전 분리됨.
호스건 동작 중 자동관수 동시 실행 가능 (하드웨어 충돌 없음).

## 현황 (2026-03-09)

- Raspberry Pi 4: 정상
- MCP23017 x2 (0x20, 0x21): 정상
- ADS1115 (0x48): 정상
- DS1307 RTC (0x68): 정상 (커널 드라이버 등록 완료)
- RS-485 토양 수분 센서 x12: 미연결 (simulation_mode=false, 연결 시 즉시 동작)
- 24채널 릴레이 보드: 정상 (구역밸브 12 + 펌프 1 + 호스건 1)
